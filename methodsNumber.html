Métodos do Objeto Number em JavaScript

toString()

O método toString() retorna um número como uma string.


let numero = 123;
let stringNumero = numero.toString();
console.log(stringNumero); // "123"

toExponential()

O método toExponential() retorna um número escrito em notação exponencial. 


let numero = 12345;
let expoNumero = numero.toExponential(2);
console.log(expoNumero); // "1.23e+4"

toFixed()

O método toFixed() retorna um número escrito com um número fixo de decimais. 


let numero = 123.456;
let fixoNumero = numero.toFixed(2);
console.log(fixoNumero); // "123.46"

toPrecision()

O método toPrecision() retorna um número escrito com um comprimento especificado. 


let numero = 123.456;
let precisoNumero = numero.toPrecision(4);
console.log(precisoNumero); // "123.5"

valueOf()

O método valueOf() retorna um número como um número.


let numero = 123;
let valorNumero = numero.valueOf();
console.log(valorNumero); // 123

Convertendo Variáveis para Números

Existem 3 métodos JavaScript que podem ser usados para converter uma variável em um número:

  • Number() - Retorna um número convertido de seu argumento.
  • parseFloat() - Analisa seu argumento e retorna um número de ponto flutuante.
  • parseInt() - Analisa seu argumento e retorna um número inteiro. 

let str = "123.45";
let num1 = Number(str);
let num2 = parseFloat(str);
let num3 = parseInt(str);
console.log(num1); // 123.45
console.log(num2); // 123.45
console.log(num3); // 123

Métodos do Objeto Number

Estes métodos pertencem ao objeto Number:

  • Number.isInteger() - Retorna verdadeiro se o argumento for um número inteiro.
  • Number.isSafeInteger() - Retorna verdadeiro se o argumento for um número inteiro seguro.
  • Number.parseFloat() - Converte uma string em um número.
  • Number.parseInt() - Converte uma string em um número inteiro.

Nota: Os métodos Number.parseInt() e Number.parseFloat() são os mesmos que os métodos globais parseInt() e parseFloat(). A finalidade é a modularização dos globais (para facilitar o uso do mesmo código JavaScript fora do navegador). 


console.log(Number.isInteger(123)); // true
console.log(Number.isSafeInteger(123)); // true
console.log(Number.parseFloat("123.45")); // 123.45
console.log(Number.parseInt("123.45")); // 123

Boa Prática de Uso

Para evitar erros, lembre-se de que os métodos do objeto Number não podem ser usados diretamente em variáveis. Eles devem ser chamados através do objeto Number: 


// Correto:
console.log(Number.isInteger(123)); // true

// Incorreto:
let x = 123;
console.log(x.isInteger()); // TypeError: x.isInteger is not a function
