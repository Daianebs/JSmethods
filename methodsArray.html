Métodos de Arrays em JavaScript

Basic Array Methods

Array length

O método length retorna o número de elementos em um array. 

let array = [1, 2, 3, 4, 5];
console.log(array.length); // 5

Array toString()

O método toString() retorna uma string representando o array. 

let array = [1, 2, 3, 4, 5];
console.log(array.toString()); // "1,2,3,4,5"

Array at()

O método at() foi introduzido no ES2022 e permite acessar elementos usando índices negativos. 

let array = [1, 2, 3, 4, 5];
console.log(array.at(2)); // 3
console.log(array.at(-1)); // 5

Array join()

O método join() junta todos os elementos do array em uma string. 

let array = [1, 2, 3, 4, 5];
console.log(array.join("-")); // "1-2-3-4-5"

Array pop()

O método pop() remove o último elemento do array e o retorna. 

let array = [1, 2, 3, 4, 5];
let lastElement = array.pop();
console.log(lastElement); // 5
console.log(array); // [1, 2, 3, 4]

Array push()

O método push() adiciona um ou mais elementos ao final do array e retorna o novo comprimento do array. 

let array = [1, 2, 3, 4];
let newLength = array.push(5);
console.log(newLength); // 5
console.log(array); // [1, 2, 3, 4, 5]

Search Methods

Array shift()

O método shift() remove o primeiro elemento do array e o retorna. 

let array = [1, 2, 3, 4, 5];
let firstElement = array.shift();
console.log(firstElement); // 1
console.log(array); // [2, 3, 4, 5]

Array unshift()

O método unshift() adiciona um ou mais elementos ao início do array e retorna o novo comprimento do array. 

let array = [2, 3, 4, 5];
let newLength = array.unshift(1);
console.log(newLength); // 5
console.log(array); // [1, 2, 3, 4, 5]

Array delete

O operador delete remove um elemento do array, mas não altera o comprimento do array. Evite usar o operador delete em arrays, pois ele não ajusta o comprimento do array e deixa elementos indefinidos. 

let array = [1, 2, 3, 4, 5];
delete array[2];
console.log(array); // [1, 2, empty, 4, 5]
console.log(array.length); // 5

Array concat()

O método concat() junta dois ou mais arrays e retorna um novo array, sem modificar os originais. 

let array1 = [1, 2, 3];
let array2 = [4, 5, 6];
let newArray = array1.concat(array2);
console.log(newArray); // [1, 2, 3, 4, 5, 6]

Array copyWithin()

O método copyWithin() copia uma parte do array para outra posição no mesmo array, sem alterar o comprimento do array. 

let array = [1, 2, 3, 4, 5];
array.copyWithin(0, 3);
console.log(array); // [4, 5, 3, 4, 5]

Array flat()

O método flat() cria um novo array com todos os elementos sub-array concatenados até a profundidade especificada, tornando mais fácil trabalhar com dados aninhados. 

let array = [1, 2, [3, 4, [5, 6]]];
let flatArray = array.flat(2);
console.log(flatArray); // [1, 2, 3, 4, 5, 6]

Array splice()

O método splice() adiciona/remova elementos de um array. O primeiro parâmetro é o índice para iniciar as mudanças, o segundo é o número de elementos a remover, e os parâmetros subsequentes são os novos elementos a serem adicionados. 

let array = [1, 2, 3, 4, 5];
array.splice(2, 1, "novo");
console.log(array); // [1, 2, "novo", 4, 5]

Array toSpliced()

O método toSpliced() retorna um novo array com alguns elementos removidos e/ou adicionados, sem modificar o array original. 

let array = [1, 2, 3, 4, 5];
let newArray = array.toSpliced(2, 1, "novo");
console.log(newArray); // [1, 2, "novo", 4, 5]
console.log(array); // [1, 2, 3, 4, 5]

Array slice()

O método slice() retorna uma cópia superficial de uma porção de um array em um novo array. O primeiro parâmetro é o índice inicial e o segundo é o índice final (não incluído). 

let array = [1, 2, 3, 4, 5];
let newArray = array.slice(1, 3);
console.log(newArray); // [2, 3]
