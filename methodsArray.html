Métodos de Arrays em JavaScript

Basic Array Methods

Array length

O método length retorna o número de elementos em um array. 


let array = [1, 2, 3, 4, 5];
console.log(array.length); // 5

Array toString()

O método toString() retorna uma string representando o array. 


let array = [1, 2, 3, 4, 5];
console.log(array.toString()); // "1,2,3,4,5"

Array at()

O método at() foi introduzido no ES2022 e permite acessar elementos usando índices negativos. 


let array = [1, 2, 3, 4, 5];
console.log(array.at(2)); // 3
console.log(array.at(-1)); // 5

Array join()

O método join() junta todos os elementos do array em uma string. 


let array = [1, 2, 3, 4, 5];
console.log(array.join("-")); // "1-2-3-4-5"

Array pop()

O método pop() remove o último elemento do array e o retorna. 


let array = [1, 2, 3, 4, 5];
let lastElement = array.pop();
console.log(lastElement); // 5
console.log(array); // [1, 2, 3, 4]

Array push()

O método push() adiciona um ou mais elementos ao final do array e retorna o novo comprimento do array. 


let array = [1, 2, 3, 4];
let newLength = array.push(5);
console.log(newLength); // 5
console.log(array); // [1, 2, 3, 4, 5]

Search Methods

Array shift()

O método shift() remove o primeiro elemento do array e o retorna. 


let array = [1, 2, 3, 4, 5];
let firstElement = array.shift();
console.log(firstElement); // 1
console.log(array); // [2, 3, 4, 5]

Array unshift()

O método unshift() adiciona um ou mais elementos ao início do array e retorna o novo comprimento do array. 


let array = [2, 3, 4, 5];
let newLength = array.unshift(1);
console.log(newLength); // 5
console.log(array); // [1, 2, 3, 4, 5]

Array delete

O operador delete remove um elemento do array, mas não altera o comprimento do array. Evite usar o operador delete em arrays, pois ele não ajusta o comprimento do array e deixa elementos indefinidos. 


let array = [1, 2, 3, 4, 5];
delete array[2];
console.log(array); // [1, 2, empty, 4, 5]
console.log(array.length); // 5

Array concat()

O método concat() junta dois ou mais arrays e retorna um novo array, sem modificar os originais. 


let array1 = [1, 2, 3];
let array2 = [4, 5, 6];
let newArray = array1.concat(array2);
console.log(newArray); // [1, 2, 3, 4, 5, 6]

Array copyWithin()

O método copyWithin() copia uma parte do array para outra posição no mesmo array, sem alterar o comprimento do array. 


let array = [1, 2, 3, 4, 5];
array.copyWithin(0, 3);
console.log(array); // [4, 5, 3, 4, 5]

Array flat()

O método flat() cria um novo array com todos os elementos sub-array concatenados até a profundidade especificada, tornando mais fácil trabalhar com dados aninhados. 


let array = [1, 2, [3, 4, [5, 6]]];
let flatArray = array.flat(2);
console.log(flatArray); // [1, 2, 3, 4, 5, 6]

Array splice()

O método splice() adiciona/remova elementos de um array. O primeiro parâmetro é o índice para iniciar as mudanças, o segundo é o número de elementos a remover, e os parâmetros subsequentes são os novos elementos a serem adicionados. 


let array = [1, 2, 3, 4, 5];
array.splice(2, 1, "novo");
console.log(array); // [1, 2, "novo", 4, 5]

Array toSpliced()

O método toSpliced() retorna um novo array com alguns elementos removidos e/ou adicionados, sem modificar o array original. 


let array = [1, 2, 3, 4, 5];
let newArray = array.toSpliced(2, 1, "novo");
console.log(newArray); // [1, 2, "novo", 4, 5]
console.log(array); // [1, 2, 3, 4, 5]

Array slice()

O método slice() retorna uma cópia superficial de uma porção de um array em um novo array. O primeiro parâmetro é o índice inicial e o segundo é o índice final (não incluído). 


let array = [1, 2, 3, 4, 5];
let newArray = array.slice(1, 3);
console.log(newArray); // [2, 3]

Array Sort Methods

Alphabetic Sort

Array sort()

O método sort() ordena os elementos de um array em ordem alfabética. Por padrão, ele ordena os elementos como strings. 


let array = ['banana', 'apple', 'orange'];
array.sort();
console.log(array); // ['apple', 'banana', 'orange']

Array reverse()

O método reverse() inverte a ordem dos elementos em um array. 


let array = ['banana', 'apple', 'orange'];
array.reverse();
console.log(array); // ['orange', 'apple', 'banana']

Array toSorted()

O método toSorted() retorna uma nova cópia do array ordenado, sem modificar o array original. 


let array = ['banana', 'apple', 'orange'];
let sortedArray = array.toSorted();
console.log(sortedArray); // ['apple', 'banana', 'orange']
console.log(array); // ['banana', 'apple', 'orange']

Array toReversed()

O método toReversed() retorna uma nova cópia do array com a ordem dos elementos invertida, sem modificar o array original. 


let array = ['banana', 'apple', 'orange'];
let reversedArray = array.toReversed();
console.log(reversedArray); // ['orange', 'apple', 'banana']
console.log(array); // ['banana', 'apple', 'orange']

Sorting Objects

Para ordenar arrays de objetos, é necessário fornecer uma função de comparação ao método sort(). 


let array = [{name: 'John', age: 30}, {name: 'Jane', age: 25}, {name: 'Jim', age: 35}];
array.sort((a, b) => a.age - b.age);
console.log(array); // [{name: 'Jane', age: 25}, {name: 'John', age: 30}, {name: 'Jim', age: 35}]

Numeric Sort

Numeric Sort

Para ordenar números, é necessário fornecer uma função de comparação ao método sort(). 


let array = [5, 2, 9, 1, 5, 6];
array.sort((a, b) => a - b);
console.log(array); // [1, 2, 5, 5, 6, 9]

Random Sort

Para ordenar um array de forma aleatória, pode-se utilizar uma função de comparação que retorne valores aleatórios. 


let array = [1, 2, 3, 4, 5];
array.sort(() => Math.random() - 0.5);
console.log(array); // [3, 5, 1, 2, 4] (ordem aleatória)

Math.min()

O método Math.min() retorna o menor valor de um conjunto de números. 


let array = [5, 2, 9, 1, 5, 6];
let min = Math.min(...array);
console.log(min); // 1

Math.max()

O método Math.max() retorna o maior valor de um conjunto de números. 


let array = [5, 2, 9, 1, 5, 6];
let max = Math.max(...array);
console.log(max); // 9

Array Iteration Methods

Os métodos de iteração de arrays operam em cada item do array.

Array forEach()

O método forEach() executa uma função em cada elemento do array. 


let array = [1, 2, 3, 4, 5];
array.forEach(element => console.log(element)); // 1 2 3 4 5

Array map()

O método map() cria um novo array com os resultados da chamada de uma função para cada elemento do array. 


let array = [1, 2, 3, 4, 5];
let newArray = array.map(x => x * 2);
console.log(newArray); // [2, 4, 6, 8, 10]

Array flatMap()

O método flatMap() primeiro mapeia cada elemento usando uma função de mapeamento e, em seguida, achata o resultado em um novo array. 


let array = [1, 2, 3, 4, 5];
let newArray = array.flatMap(x => [x, x * 2]);
console.log(newArray); // [1, 2, 2, 4, 3, 6, 4, 8, 5, 10]

Array filter()

O método filter() cria um novo array com todos os elementos que passaram no teste implementado pela função fornecida. 


let array = [1, 2, 3, 4, 5];
let newArray = array.filter(x => x > 3);
console.log(newArray); // [4, 5]

Array reduce()

O método reduce() executa uma função redutora em cada elemento do array, resultando em um único valor de retorno. 


let array = [1, 2, 3, 4, 5];
let sum = array.reduce((accumulator, currentValue) => accumulator + currentValue);
console.log(sum); // 15

Array reduceRight()

O método reduceRight() aplica uma função redutora em cada elemento do array (da direita para a esquerda) para reduzir a um único valor. 


let array = [1, 2, 3, 4, 5];
let sum = array.reduceRight((accumulator, currentValue) => accumulator + currentValue);
console.log(sum); // 15

Array every()

O método every() verifica se todos os elementos do array passam por um teste (fornecido como uma função). 


let array = [1, 2, 3, 4, 5];
let allAboveZero = array.every(x => x > 0);
console.log(allAboveZero); // true

Array some()

O método some() verifica se pelo menos um elemento no array passa por um teste (fornecido como uma função). 


let array = [1, 2, 3, 4, 5];
let someAboveThree = array.some(x => x > 3);
console.log(someAboveThree); // true

Array from()

O método from() cria uma nova instância de array a partir de um objeto iterável ou similar a array. 


let arrayLike = {0: 'a', 1: 'b', 2: 'c', length: 3};
let array = Array.from(arrayLike);
console.log(array); // ['a', 'b', 'c']

Array keys()

O método keys() retorna um novo objeto Array Iterator que contém as chaves de índice para cada elemento no array. 


let array = ['a', 'b', 'c'];
let iterator = array.keys();
for (let key of iterator) {
console.log(key); // 0 1 2
}

Array entries()

O método entries() retorna um novo objeto Array Iterator que contém pares chave/valor para cada índice no array. 


let array = ['a', 'b', 'c'];
let iterator = array.entries();
for (let entry of iterator) {
console.log(entry); // [0, 'a'] [1, 'b'] [2, 'c']
}

Array with()

O método with() retorna um novo array com o valor no índice fornecido substituído pelo novo valor, sem modificar o array original. 


let array = [1, 2, 3, 4, 5];
let newArray = array.with(2, 'novo');
console.log(newArray); // [1, 2, 'novo', 4, 5]
console.log(array); // [1, 2, 3, 4, 5]

Array Spread (...)

O operador de espalhamento (...) permite que um array seja expandido em elementos individuais. 


let array1 = [1, 2, 3];
let array2 = [4, 5, 6];
let combinedArray = [...array1, ...array2];
console.log(combinedArray); // [1, 2, 3, 4, 5, 6]
