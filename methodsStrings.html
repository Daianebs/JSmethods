Exemplos de Metódos String - JavaScript

length

Uso: Para obter o comprimento de uma string.


let str = "Hello, world!";
console.log(str.length); // 13

Boa prática: Use para validação de entrada, como limites de caracteres.

charAt()

Uso: Para obter um caractere em uma posição específica.


let str = "Hello";
console.log(str.charAt(1)); // "e"

Boa prática: Use para acessar caracteres específicos em loops ou validações.

charCodeAt()

Uso: Para obter o código Unicode de um caractere em uma posição específica.


let str = "A";
console.log(str.charCodeAt(0)); // 65

Boa prática: Útil para manipulações baseadas em códigos de caracteres, como criptografia simples.

at()

Uso: Para acessar um caractere em uma posição específica, suporta índices negativos.


let str = "Hello";
console.log(str.at(-1)); // "o"

Boa prática: Preferível a charAt por suportar índices negativos.

[ ]

Uso: Para acessar um caractere em uma posição específica.


let str = "Hello";
console.log(str[1]); // "e"

Boa prática: Similar a charAt, mas mais conciso.

slice()

Uso: Para extrair uma parte da string.


let str = "Hello, world!";
console.log(str.slice(0, 5)); // "Hello"

Boa prática: Use para manipulações de substring sem alterar a string original.

substring()

Uso: Para extrair caracteres entre dois índices.


let str = "Hello, world!";
console.log(str.substring(0, 5)); // "Hello"

Boa prática: Similar a slice, mas não aceita índices negativos.

substr()

Uso: Para extrair uma parte da string a partir de um índice, por um número específico de caracteres.


let str = "Hello, world!";
console.log(str.substr(0, 5)); // "Hello"

Boa prática: substr é legada; prefira slice ou substring.

toUpperCase()

Uso: Para converter uma string para maiúsculas.


let str = "Hello";
console.log(str.toUpperCase()); // "HELLO"

Boa prática: Útil para normalização de entrada de usuário.

toLowerCase()

Uso: Para converter uma string para minúsculas.


let str = "Hello";
console.log(str.toLowerCase()); // "hello"

Boa prática: Útil para normalização de entrada de usuário.

concat()

Uso: Para concatenar strings.


let str1 = "Hello";
let str2 = "World";
console.log(str1.concat(", ", str2, "!")); // "Hello, World!"

Boa prática: Prefira operadores de concatenação (+ ou +=) por legibilidade.

trim()

Uso: Para remover espaços em branco das extremidades da string.


let str = "  Hello  ";
console.log(str.trim()); // "Hello"

Boa prática: Use para limpeza de entradas de usuário.

trimStart()

Uso: Para remover espaços em branco do início da string.


let str = "  Hello  ";
console.log(str.trimStart()); // "Hello  "

Boa prática: Use quando apenas o início da string precisa ser limpo.

trimEnd()

Uso: Para remover espaços em branco do final da string.


let str = "  Hello  ";
console.log(str.trimEnd()); // "  Hello"

Boa prática: Use quando apenas o final da string precisa ser limpo.

padStart()

Uso: Para preencher o início da string com um caractere até um comprimento específico.


let str = "5";
console.log(str.padStart(3, "0")); // "005"

Boa prática: Útil para formatação de números ou códigos.

padEnd()

Uso: Para preencher o final da string com um caractere até um comprimento específico.


let str = "5";
console.log(str.padEnd(3, "0")); // "500"

Boa prática: Útil para formatação de números ou códigos.

repeat()

Uso: Para repetir uma string um número específico de vezes.


let str = "Hello";
console.log(str.repeat(3)); // "HelloHelloHello"

Boa prática: Útil para criar padrões ou preenchimentos repetitivos.

replace()

Uso: Para substituir parte da string por outra.


let str = "Hello, world!";
console.log(str.replace("world", "JavaScript")); // "Hello, JavaScript!"

Boa prática: Use para substituir substrings específicas; cuidado com expressões regulares.

replaceAll()

Uso: Para substituir todas as ocorrências de uma substring.


let str = "Hello, world! World, meet world.";
console.log(str.replaceAll("world", "JavaScript")); // "Hello, JavaScript! JavaScript, meet JavaScript."

Boa prática: Use para substituições globais quando não precisar de regex.

split()

Uso: Para dividir uma string em um array de substrings.


let str = "Hello, world!";
console.log(str.split(", ")); // ["Hello", "world!"]

Boa prática: Use para manipular e processar partes de uma string, como csv ou entradas de usuário.

Exemplos de Métodos de Busca de String - JavaScript

indexOf()

Uso: Para encontrar a primeira ocorrência de uma substring.


let str = "Hello, world!";
console.log(str.indexOf("world")); // 7

Boa prática: Use para encontrar a posição de uma substring específica.

lastIndexOf()

Uso: Para encontrar a última ocorrência de uma substring.


let str = "Hello, world! Hello, everyone!";
console.log(str.lastIndexOf("Hello")); // 13

Boa prática: Use quando precisar encontrar a última ocorrência de uma substring.

search()

Uso: Para encontrar a posição de uma correspondência com uma expressão regular.


let str = "Hello, world!";
console.log(str.search(/world/)); // 7

Boa prática: Use quando precisar usar expressões regulares para encontrar uma substring.

match()

Uso: Para encontrar correspondências com uma expressão regular.


let str = "Hello, world!";
console.log(str.match(/o/g)); // ["o", "o"]

Boa prática: Use para obter todas as correspondências de uma expressão regular.

matchAll()

Uso: Para encontrar todas as correspondências com uma expressão regular e retornar um iterador.


let str = "Hello, world! Hello, everyone!";
let matches = str.matchAll(/Hello/g);
for (let match of matches) {
    console.log(match);
}
// ["Hello", index: 0, input: "Hello, world! Hello, everyone!", groups: undefined]
// ["Hello", index: 13, input: "Hello, world! Hello, everyone!", groups: undefined]

Boa prática: Use quando precisar de todas as correspondências detalhadas com uma expressão regular.

includes()

Uso: Para verificar se uma string contém uma substring específica.


let str = "Hello, world!";
console.log(str.includes("world")); // true

Boa prática: Use para verificações rápidas de presença de substrings.

startsWith()

Uso: Para verificar se uma string começa com uma substring específica.


let str = "Hello, world!";
console.log(str.startsWith("Hello")); // true

Boa prática: Use para validações de prefixos de string.

endsWith()

Uso: Para verificar se uma string termina com uma substring específica.


let str = "Hello, world!";
console.log(str.endsWith("world!")); // true

Boa prática: Use para validações de sufixos de string.
